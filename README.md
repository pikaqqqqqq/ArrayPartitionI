* 选择什么排序
```
1、归并排序，最好最快时间复杂度小O(nlogn)，稳定，但是需要辅助空间O(n)，且一般都用于外排序
2、快速排序，平均时间复杂度为O(nlogn),最好时间复杂度为O(nlogn)，最坏时间复杂度为O(n2),复杂空间O(logn)~O(n),不稳定
3、优化的快速排序，使用阈值，当阈值小于47时使用简单插入排序
4、写排序算法使要给出统一的接口，如：
void Quick_Sort(ElementType A[],int N)
{
    Quicksort( A, 0, N-1 );
}
```
